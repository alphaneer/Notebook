# \*NIX中变量的用法

**变量删除**：`${变量#关键字}`, `${变量##关键字}`, `${变量%关键字}`, `${变量%%关键字}`。根据`#`,`%`和`$`的位置判断从头还是从尾进行删除

**变量替换**：`${变量/旧字符/新字符}`, `${变量//旧字符/新字符}`

**变量测试与内容替换**： 根据变量是否存在来确定是否增加或者修改变量

```shell
echo $username
# 结果为空，既可能是存在空字符变量，也有可能是不存在


```
